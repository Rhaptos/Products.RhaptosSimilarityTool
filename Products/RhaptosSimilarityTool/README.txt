RhaptosSimilarityTool

  This Zope Product is part of the Rhaptos system
  (http://software.cnx.rice.edu)

  RhaptosSimilarityTool provides a CMF tool that implements the calculation,
  storage, and retrieval of a "similarity" measure between two
  objects. 

  You can read more about the technical details in this blog entry:
   http://cnx.rice.edu/technology/techblog/Blog_Item.2004-11-15.0204
